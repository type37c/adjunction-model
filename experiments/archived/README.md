## phase2_v1_archived_20260218

**Phase 2 (v1): 内発的報酬ベースライン実験**

- **実験期間**: 2026年2月18日
- **目的**: 比較η（η_whole - η_part）を内発的報酬として、Agent Cが未知の複合形状の構造を分解的に理解できるかを検証
- **結果**: Accuracy 100%を達成したが、批判的分析により以下の問題点が明らかになった：
  1. Accuracy計算が「いずれかの部分に注目すれば100%」となっており、評価指標として不適切
  2. タスク設定が単純すぎ、エージェントは真に形状を理解することなく成功できた
  3. 価値関数がスカラ値の予測に留まり、「軌跡の予測」という理論的要請を満たしていない
  4. 状態表現が静的で、注意選択の履歴を反映できていない

- **価値**: 「比較η」のコンセプトは有望であることが示唆され、価値関数設計書v3との整合性が確認された
- **次のステップ**: Phase 2.1として、価値関数を「軌跡の予測」に再設計し、Agent CにRNN構造を導入する
