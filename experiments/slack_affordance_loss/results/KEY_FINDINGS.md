# Phase 2 Slack実験: 重要な発見

## 実験概要

- **日付**: 2026-02-14
- **エポック数**: 50
- **設定**: 再構成損失を削除、Affordance Lossのみで学習
- **目的**: 保留構造（suspension structure）の創発を観測

---

## 🎯 主要な発見

### 1. **保留構造の創発に成功**

#### η (Unit) の挙動
- **範囲**: 0.502 - 0.520
- **平均**: 0.5153
- **変化**: +3.60% (わずかに増加)
- **結論**: **ηは0に収束せず、安定して保存されている** ✅

これは、**形状の「余白」が保存されている**ことを意味する。従来のPhase 1/2/3設計では、ηは常に0に近づいていたが、Phase 2 Slackではηが保存される。

#### ε (Counit) の挙動
- **範囲**: 0.096 - 1.044
- **平均**: 0.8238
- **変化**: +983.19% (大幅に増加)
- **結論**: **εは動的に変化し、観測可能** ✅

εの大きな変動は、**アクションの曖昧性が動的に変化している**ことを示す。これは保留構造の本質的な特徴である。

---

### 2. **η-ε の強い相関**

- **相関係数**: r = 0.835 (p < 0.001)
- **回帰式**: ε = 43.92 × η - 21.81
- **R²**: 0.697

**解釈**:
- ηとεは**強く結合**している
- ηが0.01増加すると、εは約0.44増加
- **ηを保存すると、εも自動的に保存される**

この発見は非常に重要：
> **ηだけを制御すれば、εも連動して保存される可能性がある**

---

### 3. **学習とSlackの両立**

#### Affordance Loss
- **初期値**: 0.5217
- **最終値**: 0.0168
- **減少率**: 96.78%

**結論**:
- 再構成損失なしでも、アフォーダンス予測は学習可能 ✅
- ηとεを保存しながら、同時に学習が進行 ✅
- **これはPhase 2 Slackの設計目標を完全に達成している**

## 💡 Curiosity定義への示唆

### Option 1: ηのみを使用（最もシンプル）

```python
R_curiosity = η(t-1) - η(t)
```

**根拠**:
- ηは安定しており、予測可能
- 相関が強い（r=0.835）ため、ηを制御すればεも連動
- **推奨**: まずこれを試す

**利点**:
- シンプル
- 既存のActive Inference理論と整合
- 実装が容易

**欠点**:
- εの動的変化を直接活用できない

---

### Option 2: εのみを使用（探索的）

```python
R_novelty = ε(t) - ε(t-1)
```

**根拠**:
- εは変動が大きく、探索的
- 新奇性の指標として有用

**利点**:
- 動的な変化を捉えられる
- 探索行動を促進

**欠点**:
- 変動が大きいため、不安定な可能性
- ηとの関係を無視

---

### Option 3: 別々の報酬として定義（推奨）

```python
R_understanding = η(t-1) - η(t)  # 理解の深化
R_exploration = ε(t) - ε(t-1)   # 探索の拡張

R_intrinsic = α × R_understanding + β × R_exploration
```

**根拠**:
- ηとεは異なる性質を持つ
- 別々に扱うことで柔軟性が高まる
- 重みα/βで調整可能

**利点**:
- 最も柔軟
- ηとεの両方の特性を活用
- 実験的に最適な重みを探索可能

**欠点**:
- 複雑
- ハイパーパラメータ調整が必要
